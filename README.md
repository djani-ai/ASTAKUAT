# Astakuat PAC GP Ansor Brondong

## Deskripsi

Aplikasi Astakuat adalah platform web yang dikembangkan untuk mendukung kegiatan PAC GP Ansor Brondong. Aplikasi ini dirancang untuk memfasilitasi manajemen data, komunikasi, dan aktivitas organisasi.

## Fitur Utama

- Manajemen anggota
- Pengelolaan acara
- Sistem komunikasi internal
- Dashboard informasi

## Instalasi

- composer install
- cp .env.example .env
- php artisan key generate
- php artisan migrate:fresh --seed
- php artisan make:filament-user

## Penggunaan

- Akses aplikasi melalui URL utama.
- Login menggunakan kredensial yang diberikan.
- Navigasi melalui menu untuk mengakses fitur-fitur.
